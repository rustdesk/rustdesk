# Tabby logo concepts

This folder contains the retried concept sheet for the Tabby logo process described in `logo-prompt.md`, with the tone adjusted toward the referenced techy, high-end, soft, cool app icon style.

## Concept sheet

- [`tabby-logo-concepts.svg`](./tabby-logo-concepts.svg) — three small-thumbnail directions with 16px checks.

## Directions shown

1. **Frost Paw** — a premium paw made from keycap geometry. The soft glassy surfaces keep it high-end while the extra-control toes and hidden Tab/plus shape carry the “extra paws/powers” double meaning.
2. **Command Eye** — a tech-forward cat eye as command focus. The whisker vectors feel like precision input paths or shortcuts, keeping the cat reference sharp rather than cute.
3. **Mod Cat** — a restrained cat silhouette with icy depth and negative-space modifier keys. It reads as a developer-tool badge first, cat metaphor second.

## Next step
