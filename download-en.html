🚀 Installation & Usage Guide

1

Download and Install

Click the download button above, then run the installer and follow the setup wizard to complete installation.

2

Set Permanent Password

When you first open RustDesk, find the "Password" setting on the right side of the main interface and set a permanent password. Recommended: use a strong password (at least 12 characters with uppercase, lowercase, and numbers).

3

Get Your ID

You can see your unique ID at the top of the RustDesk main interface (e.g., 123456789). Share this ID and password with people who need to remotely connect to you.

4

Connect to Other Computers

Enter the other person's ID in the "Remote Desktop" input box, click "Connect", and enter their password to establish a remote connection.
