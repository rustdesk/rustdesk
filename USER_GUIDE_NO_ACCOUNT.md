# RustDesk 使用指南 - 无需账号模式

## 🚀 快速开始（无需注册）

### 第一步：安装客户端
运行 `RustDesk_Cislink_Setup.exe` 安装客户端（已预配置服务器）

### 第二步：设置永久密码

1. 打开 RustDesk 客户端
2. 在主界面右侧找到 "密码" 设置
3. 点击 "设置密码" 或编辑图标
4. 输入一个永久密码（建议使用强密码）
5. 确认密码

**注意**: 这个密码是供其他人远程连接到你的电脑时使用的

### 第三步：获取你的 ID

在 RustDesk 主界面顶部可以看到：
```
你的桌面 ID: 123456789  (示例)
密码: ********
```

### 第四步：连接到其他电脑

1. 在 "远程桌面" 输入框输入对方的 ID
2. 点击 "连接"
3. 输入对方设置的永久密码
4. 连接成功！

| 特性 | 无账号模式 | 使用账号 |
|------|-----------|---------|
| 设置复杂度 | ⭐ 简单 | ⭐⭐ 需配置 |
| 连接方式 | ID + 密码 | 通讯录 |
| 适用场景 | 点对点连接 | 管理多台设备 |
| 推荐 | ✅ 自托管推荐 | ⚠️ 需额外配置 |

## 🔐 安全建议

### 永久密码设置建议：
- 长度至少 12 位
- 包含大小写字母、数字、特殊字符
- 不要使用常见密码
- 定期更换

### 示例强密码：
```
Cislink@2025!Rd
MySecure#Pass123
```

## 💡 常见场景

### 场景 1: 远程帮助家人
1. 告诉家人你的 ID
2. 告诉家人你的永久密码
3. 家人输入 ID 连接
4. 输入密码后即可控制你的电脑

### 场景 2: 访问公司电脑
1. 在公司电脑设置永久密码
2. 记下公司电脑的 ID
3. 在家输入公司电脑 ID
4. 输入密码连接

### 场景 3: 技术支持客户
1. 客户安装 RustDesk
2. 客户告诉你 ID 和临时密码
3. 你输入 ID 连接
4. 提供技术支持

---

## ⚠️ 注意事项

1. **ID 是唯一的**: 每台设备安装后自动生成，不会改变
2. **密码可以修改**: 随时可以更改永久密码
3. **无需互联网账号**: 完全私有，无需注册任何账号
4. **数据不经过第三方**: 所有连接通过你的服务器 (hbbs.cislink.nl)

## 🔧 高级功能

### 临时密码
- 每次启动 RustDesk 都会生成一个临时密码
- 临时密码会定期更换
- 适合一次性访问

### 一次性密码
- 可以设置只能使用一次的密码
- 使用后立即失效
- 适合临时技术支持

### 无人值守访问
- 设置永久密码后
- 即使电脑无人也可以远程连接
- 适合远程管理服务器

---

## 📞 故障排查

### 无法连接？
1. 检查网络连接
2. 确认 ID 输入正确
3. 确认密码输入正确
4. 检查防火墙设置

### 忘记密码？
- 在被控端电脑上可以随时重新设置
- 密码存储在本地，无法远程找回

---

**总结**: 使用自托管 RustDesk 服务器时，**不需要注册账号**！只需要 ID 和密码即可实现安全的远程连接。
