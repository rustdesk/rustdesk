# Tabby Changelog — May 3–10, 2026

## Multi-Session Support
- **Multi-session architecture** — run up to 5 concurrent remote sessions simultaneously. Sessions are managed via a `SessionRegistry` with an `IndexedStack` host that keeps all sessions alive. Switch between sessions with a badge picker. Background sessions enter view-only mode to save resources and resume on focus.

## Canvas & Keyboard
- **Keyboard pan** — when the iOS keyboard appears, the remote canvas pans upward (40% of keyboard height) instead of shrinking, keeping the session content visible.
- **Canvas zoom memory** — last zoom level is saved per connection and restored on reconnect; opt-in toggle in Settings.
- **Display memory** — last active display is saved per peer and restored on reconnect; opt-in toggle in Settings.

## Chat Overlay
- **Full-screen textarea** — the chat overlay is now a full-screen multi-line text input. Tap send to close and transmit; no more command history UI.

## Floating Macro Bar
- **Floating vertical macro bar** — draggable, scrollable, collapsible macro strip anchored to the right edge.
- **Rectangle window macros** — maximize, half-width, and a 4-corner submenu for snap-to-quadrant layouts.
- **F-key aliases** — F1–F12 mapped via VK_* compat + Option+Enter+F12 macro.
- **Additional macros** — Ctrl+V, Cmd+V, Cmd+Space (Telegram), git commit shortcut.
- **Macro bar anchor fix** — bar opens upward, anchored above the custom strip.
- **Scrollable macro bar** — bar height-capped and scrollable for small screens.

## PowerStrip
- **Multi-monitor display switcher** — button in PowerStrip that reactively appears when a multi-monitor session is detected; cycles through displays.
- **Settings tab** — added Settings as a dedicated nav tab.
- **typeString key + Y shortcut** — new KeyType for typing strings added to the default strip.

## Keep-Awake Idle Timeout
- **Idle timeout dropdown** — Settings option to auto-disable keep-awake after 30s / 1m / 5m / 15m / 30m / custom; resets on any interaction.
- **Performance fix** — idle timeout seconds are cached at connect time; FFI is not called on every input event.

## Import from RustDesk
- **Full import flow** — Rust FFI parses `peers/*.toml` and `RustDesk2.toml` from the app's root folder; Dart bridge orchestrates the import.
- **5-slide walkthrough modal** — step-by-step guide with both Copy→Paste and Share→Save paths for transferring files from the RustDesk app.
- **Conflict resolution dialog** — per-peer Override/Skip/All options when imported peers already exist.
- **Settings UI** — "Import from RustDesk" and guide rows added to Settings page.
- **Guide fix** — corrected folder reference from empty `data/` subfolder to the actual root folder.

## UI Polish
- **Tabby logo** — transparent-background logo used consistently across all icon placements (connect screen, login form, app bar).
- **Dialog centering** — password and overlay dialogs are now correctly centered above the keyboard on iOS.
- **BotToast overlay** — darkened loading overlay background to `black54` for better visibility.
